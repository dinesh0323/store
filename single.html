Product Brief Description

This system allows for outstanding operator control, with the trigger-control system that has a power switch in both the plunge-base's and the fixed-base's handle. The plunge base's Afterlock microfine bit-depth adjustment at any plunge setting allows for precision, with an adjustable turret stop makes it easy to set any cumulative depth. With a smooth plunge action and a spring-lock plunge lever, this router gets accurate results. The fixed base has microfine bit-depth adjustment, with a continuous depth range of 1-5/8 in, allowing for accurate and fast changes, while its precision-centering design allows the bit to be centered with the sub-base or optional template guides. The router has a 15 Amp motor and adjustable speed settings. The variable-speed dial allows the operator to match the speed to the material. Constant Response circuitry delivers a constant speed throughout the cut. Users can glide along the intended cutline using dovetail fixtures, templates, jigs and other guidance devices.
